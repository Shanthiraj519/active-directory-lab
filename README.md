# Active Directory Lab 🚀

This repository documents my hands-on learning and implementation of Active Directory using a home lab environment.

## 🧱 Lab Environment

* **Host Machine:** Windows 11 (Hyper-V)
* **Virtualization:** Proxmox
* **Domain Controllers:**

  * dc-prox-01 (Primary Domain Controller)
  * dc-prox-02 (Additional Domain Controller)
  * dc-azu (Azure Domain Controller)
* **Client Machine:** Windows 10/11

## ⚙️ What This Repo Covers

* Active Directory Domain Services (AD DS)
* Domain setup and configuration
* DNS configuration and troubleshooting
* User and Group Management
* Group Policy (GPO) implementation
* PowerShell automation for AD tasks
* Real-world troubleshooting scenarios
* Basic Active Directory security concepts

## 📁 Repository Structure

* `setup/` → Domain setup, DNS, DC configuration
* `users-groups/` → User and group management scripts
* `group-policy/` → GPO configurations and use cases
* `powershell/` → Automation scripts
* `troubleshooting/` → Issues faced and solutions
* `security/` → AD security practices

## 🛠️ Key Learning Areas

* Creating and managing users using GUI & PowerShell
* Configuring and troubleshooting Group Policies
* Managing OU structure and permissions
* Diagnosing replication and login issues
* Implementing basic security best practices

## 🎯 Objective

To build strong, practical knowledge of Active Directory and align it with real-world system administration and Azure Security concepts.
